# Premium Food Ordering Website

A modern, premium animated food ordering website with rich UI, smooth transitions, and attractive food visuals.

## Features
- Clean, elegant layout with warm food-inspired colors.
- Glassmorphism effects and smooth hover animations.
- Fully responsive design.
- Animated Menu and Hero sections.
- Parallax effects and scroll-triggered animations.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Minimal/Clean)

## Design Palette
- Cream: `#FFF8F0`
- Orange: `#FF7F32`
- Deep Red: `#8B0000`
- Dark Brown: `#2D1B08`
